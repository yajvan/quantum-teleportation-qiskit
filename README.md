# Quantum Teleportation Simulator using Qiskit

## Overview
This project demonstrates Quantum Teleportation using Qiskit and Python.

The simulation includes:
- Random quantum state generation
- Quantum entanglement
- Bell-state measurement
- Quantum teleportation protocol
- Statevector verification using Qiskit Aer

## Technologies Used
- Python
- Qiskit
- NumPy
- Qiskit Aer Simulator

## Project Structure

- `main.py` → Runs the complete simulation
- `state_generator.py` → Generates random quantum states
- `entanglement.py` → Creates entangled Bell pairs
- `teleportation.py` → Implements teleportation protocol
- `verification.py` → Verifies final quantum state

## How to Run

Install dependencies:

```bash
pip install qiskit qiskit-aer numpy
```

Run the project:

```bash
python main.py
```
